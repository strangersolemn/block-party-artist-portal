HOW IT WORKS

THE FULL DMT MECHANICS
1 · THE ELEMENT — SUPPLY FILTER
1ee7.1337.12.element · inscribed 22 Apr 2026 · id 3b6646a2…e80d7e0i0

Claims the pattern "1337" in field 12 (difficulty). A block qualifies as an Acid Frog only if its difficulty header field contains the substring 1337. Because the difficulty value is locked for all 2,016 blocks of an epoch, one qualifying difficulty = exactly 2,016 qualifying frogs.

In Bitcoin's 15-year history that's happened exactly once: epoch 941472–943487, 20 March–3 April 2026, difficulty 133793147307542.
2 · WHY FIELD 12 — THE PIONEER PLAY
Fields currently indexed by Trac hold every DMT project ever shipped:
· Field 11 (bits) — Natcats (3b), Royals (cc)
· Field 10 (nonce) — handful of projects
· Field 4 (height) — dmt-cash, aurumdraconisnats, etc.

Field 12 (difficulty): zero projects. Acid Frogs is the first element ever in this field. The narrative is equal to the execution — "first project in the difficulty field" + "first time Bitcoin's difficulty contained 1337" is a double-first nobody else can claim.

Field 12 also has the same 2016-block clustering mechanic as field 11 — difficulty is recalculated only at retarget. So we get the "generation" magic that made Natcats work, but in untouched territory.
3 · TRAIT DERIVATION — DMT-PURE
Every frog's art is deterministic from its block's merkleroot (64 hex chars = 256 bits). We split the merkleroot into bytes and pick traits via weighted lookup tables. No PRNG library, no external state — the renderer reads block data once and composes.

byte[0] → frog colour  (Green 70% / Blue 10% / Red 8% / Pink 7% / Purple 5%)
byte[1] → head   (1 of 44)
byte[2] → eyes   (1 of 25)
byte[3] → clothing (1 of 12)
byte[4] → mouth  (1 of 2)
byte[5] → smoke  (~22% chance, 1 of 2)
byte[6] → background (1 of 8)

Same mechanic Natcats uses for height-digit colour picks, just upgraded from 10-value digits to 256-value bytes — lets us have weighted rarity (Legendary Purple frog: 13/256 = ~5% = ~100 pieces; Green Common: 179/256 = ~70% = ~1,400 pieces) without leaving DMT purity.
4 · AUDIO — GENERATIVE 303 FROM NONCE
Each frog plays a unique 16-step acid bassline in tech-house at 123 BPM. The nonce (32 bits) is the seed:

nonce binary = 32 bits → 16 steps × 2 bits per step
  00 = rest
  01 = play root note
  10 = play root + octave
  11 = play root with slide

nonce[0] → root note (16 values → c2, d2, eb2, e2, f2, g2, ab2, a2, bb2, b2, c3, d3, eb3, f3, g3, a3)
nonce[1] → filter cutoff base (0-3000Hz)


Drum pattern is fixed (four-on-floor kick + clap on 5 & 13 + off-beat hats — classic tech house). The 303 line on top is where every frog's individuality lives.

6 · UNRELEASED TRAITS — LIVES FOR DECADES
Bitcoin's difficulty only grows. Every few years a future difficulty number will happen to contain 1337 again, adding 2,016 more pieces and unlocking traits that were coded into the renderer but never triggered by the April 2026 epoch. Examples:

· "Millennium" trait fires only if difficulty > 1,000T (unlocks circa 2035)
· "Halving Echo" fires only on blocks within 24h of a halving
· "Cosmic" fires only when block mined at exactly 13:37 UTC

This is the same mechanic Natcats advertises as "autonomous evolution." Holders get speculative upside that the collection is alive long after mint.
